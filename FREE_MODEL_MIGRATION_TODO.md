# Free Model Migration TODO

이 문서는 ConceptBug의 번역/분석/이미지생성 기능을
무료 모델 우선 구조로 전환하기 위한 작업 목록입니다.

## Goal

- API 키가 없어도 핵심 기능(번역/분석/이미지 생성)을 기본 사용 가능하게 만든다.
- API 키가 있으면 키 형식에 따라 provider를 자동 라우팅한다.
- 모델별 지원 해상도만 선택 가능하도록 UI를 동기화한다.
